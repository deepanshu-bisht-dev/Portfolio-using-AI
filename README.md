# 🌐 Portfolio — Built with AI (Google AI Studio)

A modern, dark "cyber-tech" themed personal portfolio website, generated and built using Google AI Studio with Gemini. Built to explore how far AI-assisted frontend development can go — from layout to animation to copy.

---

## 📌 About This Project

This portfolio was scaffolded and iterated on using Google AI Studio, then exported as a standalone React app. It's an experiment in AI-assisted UI design — a fast way to go from idea to a fully animated, responsive personal site.

- 🎓 B.Tech CSE (AI/ML Specialization), Future University
- 🛠️ Focus: AI-assisted frontend development and modern UI/UX
- 📈 Goal: A polished personal site to showcase projects and skills

---

## ✨ Sections

- **Hero** — intro, headline stats, and call-to-action buttons
- **Technical Expertise** — skill cards (Frontend, Backend, API & AI Integration)
- **About** — short bio with highlight stats
- **Featured Projects** — large showcase cards for top projects
- **All Projects** — grid of additional project cards
- **Contact** — contact details + message form

---

## 🧰 Tech Stack

- **Framework:** React 19 + Vite 6
- **Styling:** Tailwind CSS 4
- **Animation:** Motion (formerly Framer Motion)
- **Icons:** Lucide React
- **AI:** Google Gemini API (`@google/genai`)
- **Backend (scaffolded):** Express + better-sqlite3 + dotenv

---

## 🚀 Getting Started

**Prerequisite:** Node.js

```bash
git clone https://github.com/deepanshu-bisht-dev/Portfolio-using-AI.git
cd Portfolio-using-AI
npm install
```

Set your Gemini API key — copy `.env.example` to `.env.local` and fill it in:

```bash
cp .env.example .env.local
```

```env
GEMINI_API_KEY="your-gemini-api-key"
```

Run locally:

```bash
npm run dev
```
